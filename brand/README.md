# Tripfluent brand assets

Portable SVGs — text is converted to **outlines**, so they render identically anywhere
(no fonts required). All have **transparent backgrounds** except the app icon.

| File | What it is |
|------|------------|
| `wordmark.svg` | The "Tripfluent" logotype (Trip = Plus Jakarta 800 navy, fluent = Playfair italic blue) |
| `lighthouse.svg` | The lighthouse mark, on its own |
| `lockup.svg` | Wordmark + lighthouse, side by side (the primary logo) |
| `icon-tf.svg` | The "Tf" app-icon monogram on a cream tile |

## Colors
- Navy (primary): `#1c275e`
- Blue (secondary): `#4a7fc1`
- Gold (accent): `#d8b713`
- Cream (background): `#fcfcf3`
- Ink (text): `#050505`

## Type
- Display / headings + wordmark "Trip": **Plus Jakarta Sans** (800)
- Wordmark "fluent": **Playfair Display** Italic (500)
- Body: **Inter**

Need a PNG at a specific size, a white/mono version, or a horizontal-vs-stacked variant? Just ask.
