# Spanish Trip Trainer 🇪🇸

A mobile-first PWA for learning travel Spanish, Barcelona-style. Travel phrases
first, breadth across topics over grammar depth. Streaks, XP, and tongue-in-cheek
rewards.

Pure static files — no build step. Open `index.html` or host anywhere static.

- `curriculum.js` — all lessons & phrases (edit here to add content)
- `app.js` — exercise engine (match / multiple-choice / tap-to-build)
- `messages.js` — the funny reward lines
- `styles.css` — UI
- `sw.js` + `manifest.webmanifest` — offline + installable-to-home-screen
